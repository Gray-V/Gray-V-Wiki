Overview

Eve is a farming simulation game with adventure mechanics. Developed in Unity using Pro Motion NG, Procreate, and Fmod.

Key Features

  • Farming simulation with dynamic crop growth.
  • Adventure mechanics like quests and exploration.
  • Pixel art with 2.5D rendering for an immersive experience.

Development Details

This game was developed as part of a senior project using:

  • Unity for game mechanics and physics.
  • Pro Motion NG for pixel art assets.
  • Fmod for adaptive audio and sound effects.
  • Procreate for detailed character portraits.
