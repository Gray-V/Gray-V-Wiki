Sharkitect Game Engine

Overview

Sharkitect is a physics-based game engine developed as an independent study project by Grayson von Goetz und Schwanenfliess and Matthew Lee. Written in C, it utilizes vector mathematics to handle user-interactive collision calculations, demonstrating a deep understanding of low-level programming and physics simulation.

Key Features

  • Physics engine capable of real-time collision detection and response.
  • Developed in C, showcasing proficiency in low-level programming.
  • Utilizes vector mathematics for accurate physics simulations.
  • OpenGL integration for rendering graphics.
